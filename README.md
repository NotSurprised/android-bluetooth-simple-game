# Android Bluetooth Game - 皮卡丘打排球藍牙連線版

透過藍牙與對手進行一場排球比賽！

這是一套我們透過遊戲製作的原理以及 Android 的藍牙模組製作的 Android 手機遊戲，遊戲的主題仿製了在桌面系統上相當流行的皮卡丘打排球遊戲。
